# Skeleton Tracker (Webcam + Optional Phone)

![Skeleton Tracker Screenshot](https://i.postimg.cc/Mn9NCj8R/Screenshot-2025-11-24-140450.png)

**Real-time skeleton tracker using your PC webcam (phone optional).** Detects head, neck, torso, arms, hands, legs, and feet, drawing connected cyan lines and red landmarks. Fully browser-based, selectable webcams, smooth tracking, and lightweight.

---

## Features

- Real-time skeleton detection in the browser  
- Tracks head, neck, torso, arms, hands, legs, and feet  
- Red landmarks and cyan lines connecting joints  
- Selectable webcams on your PC  
- Optional phone camera via virtual webcam or IP camera  
- Lightweight and easy to run — no extra hardware required  

---

## Installation

1. Clone this repository:

```bash
git clone https://github.com/houdineh/RealTimeSkeletonTracker.git
cd RealTimeSkeletonTracker
````

2. Open `index.html` in a browser (Chrome/Edge recommended).
3. Allow camera access when prompted.

---

## Usage

1. Select your PC webcam from the dropdown.
2. (Optional) Use a phone camera via a virtual webcam or IP camera.
3. The video will appear with the real-time skeleton overlay.
